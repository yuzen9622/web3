Introduce

網頁前端程式三大元素：HTML、CSS、JAVASCRIPT

網頁是由前端、後端兩大部分組成，前端採集輸入資訊，後端進行資訊處理。 簡言之，我們平常看到的網頁就是前端，而後端程式則構成了後台管理系統。 今天我們要介紹的只有網頁前端設計，由三大元素構成：HTML、CSS、JavaScript。

  • HTML ：網頁內容的描述語言，負責建立網頁的主結構。

  • CSS ：網頁外觀形態的描述語言，負責美化網頁。

  • JavaScript ：一門函式先行的直譯式程式語言，經常用在呈現網頁動態效果。負責描述網頁如何與用戶互動。

透過HTML以及CSS的相互合作，就能將網頁結構化，做成我們希望的視覺呈現方式，再藉由JavaScript， 讓我們可以在只能閱覽文字與影像的網頁中，加入動態效果並與使用者互動。

網頁前端三大框架：React、Angular、Vue

網頁是由前端、後端兩大部分組成，前端採集輸入資訊，後端進行資訊處理。 簡言之，我們平常看到的網頁就是前端，而後端程式則構成了後台管理系統。 今天我們要介紹的只有網頁前端設計，由三大元素構成：HTML、CSS、JavaScript。

  • React:通過簡潔的API實現了虛擬DOM和組件的高效更新

  • Angular:強調數據的視圖分離，能夠快速實現數據變化的反應

  • Vue: 使用了單向數據流的方式進行數據綁定

但共通點是，三大框架都漸漸走向組件化的開發思想，以實現高內聚、低耦合的目標

當代 JavaScript 框架的問世，讓構建高度動態的互動式應用，變得簡單許多。框架是個針對軟體構建，提供完整解決方案的函式庫。這些選項能讓應用程式，開始能預測和同質化。可預測性讓軟體能擴展到巨大的規模時依舊能維護；可預測性和可維護性則對軟體的健康和長壽至關重要。
